# Сортировка перемешиванием (Cocktail Sort / Шейкерная сортировка)

## Описание проекта
Программа на языке C для сортировки массивов целых чисел алгоритмом шейкерной сортировки. 
Проект выполнен в рамках летней практики.

## Состав команды
- Студент 1: Реализация алгоритма сортировки
- Студент 2: Работа с файлами (CSV)
- Студент 3: Меню, тестирование, документация (это ты)

## Функциональность
- Загрузка массива из файла формата CSV
- Генерация случайного массива с заданным диапазоном
- Сортировка шейкером
- Вывод времени работы и количества перестановок
- Сохранение результата в CSV

## Сборка и запуск
Для компиляции используйте GCC или Visual Studio:
gcc main.c -o cocktail_sort.exe
./cocktail_sort.exe
